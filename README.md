# Tourland

Minimal React website for **Tourland** (Sri Lanka tourism company).

## Local dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push this repo to GitHub
2. In Vercel → **New Project** → Import the repo
3. Framework: **Vite**
4. Build Command: `npm run build`
5. Output Directory: `dist`
